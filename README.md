## 📌 Project Overview

This project follows a complete analytics pipeline — from raw data to business decisions:

| Phase | Tool | What Was Done |
|-------|------|---------------|
| 🐍 Phase 1 | Python & Pandas | Data cleaning, EDA, feature engineering, MySQL integration |
| 🗄️ Phase 2 | MySQL Workbench | 10 business questions answered using SQL |
| 📊 Phase 3 | Power BI | Interactive dashboard with live MySQL connection |
| 📄 Phase 4 | Claude AI | Professional report & presentation generated |

---

## 🔍 Key Insights

- 💰 Male customers generated **2x more revenue** than female customers
- 👶 **Young Adults** are the highest revenue-contributing age group
- 📬 Only **27% subscribe** despite 73% being repeat buyers — major conversion opportunity
- 🎯 Top discounted products carry **~50% discount rates** — margin review needed
- 🚚 Express shipping users spend **~$2 more** per order than Standard

## 🚀 How to Run

1. Clone the repository
2. Run `Project_Notebook.ipynb` in Jupyter
3. Import cleaned data to MySQL using the notebook
4. Open `customer_behavior_analysis.sql` in MySQL Workbench
5. Connect Power BI to your MySQL `customer_behavior` database
